# ライブラリ

ライブラリ・コンポーネントの選定方針をまとめる。

| 利用対象 | ライブラリ | 理由 |
| --- | --- | --- |
| UI コンポーネント | shadcn/ui | デフォルトでアクセシビリティ対応かつ質の高いデザインが用意されており、さらにコードをフルカスタマイズできるため細かい要件にも対応できる |
| 日付操作 | date-fns | 関数単位でインポートでき、バンドルサイズを小さく保てる |
| バリデーション | valibot | モジュラー API を当初から設計思想として作り込んでおり成熟度・一貫性が高く、必要な機能だけを取り込めてバンドルサイズも小さく保てる（同種の zod-mini は後発の変種） |
